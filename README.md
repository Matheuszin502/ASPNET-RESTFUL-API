## 📖 Descrição
Este projeto se trata de uma API RESTful feita em ASP.NET para uma livraria virtual.

### 💻 Tecnologias
- C#
- ASP.NET
- SQL Server
- Docker
- Swagger UI

### ⚙️ Operações que a API suporta
A API suporta todas as operações de CRUD para um banco de dados SQL Server, permitindo também navegar entre os endpoints usando links
HATEOAS.

### 🚀 Como Rodar a API
Basta usar o comando docker-compose up na pasta em que está o docker-compose.yaml, para saber o endereço dos endpoins, pode se consultar
o Swagger UI.
